1부. 재료 준비: 데이터의 기초와 리스트
01. 안녕, 파이썬!: 가장 친절한 언어, 파이썬 설치와 첫 실행.

02. 이름표 붙이기: 변수 정의와 상수의 관례(대문자 이름표).

03. 다양한 재료: 숫자, 문자열, 그리고 참/거짓을 다루는 불리언(Boolean).

04. 강력한 바구니: 리스트(List) 기초와 for ... in 반복문.

05. 모눈종이 게임판: 리스트 안에 리스트를 넣는 2차원 리스트로 격자판 만들기.

2부. 기본 동작: 소통과 논리의 시작
06. 도화지에 그리기: print() 함수와 ANSI 코드로 터미널 화면 제어하기.

07. 컴퓨터와 대화하기: input()으로 사용자 입력받기와 데이터 타입 변환.

08. 스마트한 판단: if-elif-else 조건문으로 충돌 판단 로직 설계하기.

09. 반복의 힘: while 루프를 이용한 실시간 게임 엔진 뼈대 만들기.

3부. 논리의 조립: 함수와 참조의 이해
10. 마법의 주문 (Function): 반복되는 기능을 함수로 쪼개고 재사용하기.

11. 파이썬의 비밀: 객체 참조 (Reference):

"왜 리스트를 함수에 넣으면 원본이 바뀔까?"

가변(Mutable) 객체와 불변(Immutable) 객체의 결정적 차이 이해하기.

12. 부품 설계도 (Class): 블록의 x, y 좌표와 모양을 하나로 묶는 클래스 기초.

13. 뒷정리와 예외 처리: try-except-finally로 예기치 못한 오류 대처하기.

4부. 설계의 기술: 객체 지향 테트리스 (OOP)
14. 스스로 행동하는 객체: 클래스 메서드(self)와 속성 관리하기.

15. 부모와 자식 블록: **상속(Inheritance)**을 이용해 다양한 모양의 블록을 효율적으로 만들기.

16. 코드 모듈화: import를 사용해 파일을 나누고 깔끔하게 관리하기.

정보를 숨기는 법: 파이썬의 언더바(_) 관례 이해하기.

5부. 시간의 마법: 멈추지 않는 세상 (Concurrency)
17. 일정한 리듬: time 모듈로 중력(자동 하강) 구현하기.

18. 여러 일 동시에 하기: threading 모듈을 이용해 입력 대기와 하강을 동시에 처리하기.

19. 안전한 데이터 통신: queue 모듈을 이용해 스레드 간에 안전하게 신호 주고받기.

20. 최종 프로젝트: 나만의 파이썬 테트리스 완성 및 리팩토링.

부록 1. 현대적인 파이썬 관리자: uv 깊게 보기
왜 uv를 썼을까요?: pip와 venv를 몰라도 되는 이유.

프로젝트의 심장: pyproject.toml 읽어보기.

코드 예쁘게 만들기: Ruff를 이용한 포매팅과 린터 설정 (자동으로 오타와 줄 바꿈 고치기).

부록 2. 범인 찾기: 디버깅(Debugging)의 기술
중단점(Breakpoint): breakpoint()로 프로그램 멈추고 살펴보기.

에러 메시지 읽기: 컴퓨터가 보내는 구조 신호(Traceback) 해석하기.

부록 3. 성능의 돋보기: 프로파일링(Profiling)
cProfile 맛보기: 내 게임에서 가장 느린 구간을 데이터로 찾아내기.
