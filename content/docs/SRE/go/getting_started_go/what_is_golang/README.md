# Go 언어 시작하기

## Go 언어란 무엇인가

`Go 언어`는 2009년 구글(Google)에서 개발한 오픈 소스 프로그래밍 언어입니다. 로버트 그리머, 롭 파이크, 켄 톰슨 등 컴퓨터 과학계의 거장들이 주도하여 만들었으며, 복잡한 시스템을 효율적으로 구축하기 위해 설계되었습니다.

구글의 거대한 시스템 내에서 기존 언어(C++, Java 등)들이 가졌던 다음과 같은 문제점들을 해결하고자 탄생했습니다.

* 느린 컴파일 속도: 대규모 프로젝트의 빌드 시간을 획기적으로 단축.
* 복잡한 의존성: 정적 링크를 통해 실행 파일 하나만으로 배포 가능하게 함.
* 멀티코어 활용의 어려움: 병렬 처리를 언어 차원에서 쉽게 지원하도록 설계.

또한 다음과 같은 특징을 가지고 있습니다.

**심플한 문법 (Simplicity)**

`C`언어를 계승하여 `Go 언어`는 매우 작은 프로그래밍 언어입니다. 키워드가 약 25개 내외로 매우 적어 학습 곡선이 낮습니다.

'한 가지 문제는 한 가지 방법으로 해결한다'는 철학을 가지고 있어 코드 가독성이 뛰어납니다.

**강력한 동시성 (Concurrency)**

`Go 언어`는 매우 강력한 동시성 모델을 제공합니다. 
* Goroutine(고루틴): OS 스레드보다 훨씬 가벼운 논리적 스레드로, 수만 개를 동시에 실행해도 메모리 부하가 적습니다.
* Channel(채널): 데이터를 안전하게 주고받으며 메모리를 공유하지 않고 통신하는 방식을 취합니다.

**빠른 성능과 정적 타입**

컴파일 언어로서 C/C++에 근접한 실행 속도를 냅니다. 또한, 강력한 타입 체크로 런타임 에러를 줄여줍니다.

**가비지 컬렉션 (Garbage Collection)**
메모리 관리를 자동으로 수행하여 개발자의 부담을 덜어주면서도, 지연 시간(Latency)을 최소화하도록 최적화되어 있습니다.

## 왜 Go 언어인가?
"Go 언어는 파이썬의 생산성과 C++의 성능 사이에서 완벽한 균형을 찾는 언어입니다."

단순히 언어가 빠른 것을 넘어, **"팀 전체의 개발 속도와 유지보수 효율"**을 극대화하는 데 목적이 있습니다. 배포가 간편하고 협업 시 코드 스타일 충돌이 적어 대규모 팀 프로젝트에 매우 적합합니다.

`Go 언어`는 최근 다음과 같은 분야에서 활발하게 사용되고 있습니다.

* 클라우드 & 인프라: `Docker`, `Kubernetes`, `Terraform` 등 현대 클라우드 도구의 대부분이 `Go 언어`로 작성되었습니다.
* 백엔드 API 서버: 빠른 성능과 동시성 덕분에 고성능 마이크로서비스(MSA) 구축에 최적입니다.
* 네트워크 도구: 분산 시스템, 프록시 서버, 메시징 큐 서비스 개발에 많이 쓰입니다.
* CLI 도구: 실행 파일 하나만 배포하면 되기 때문에 터미널용 유틸리티 개발 시 인기가 높습니다.

## Go 언어 설치하기 

각 플랫폼 별로 다음

### Windows

```posh
$ winget install GoLang.Go
```

### MacOS

```bash
$ brew install go
```

### 리눅스 (Debian 계열)

```bash
$ sudo apt update
$ sudo apt install golang-go
```
