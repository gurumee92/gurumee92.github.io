# Go 언어 설치하기 

## Go 언어 설치하기

각 플랫폼 별로 다음과 같이 설치할 수 있습니다.

### Windows

```posh
$ winget install GoLang.Go
```

### MacOS

```bash
$ brew install go
```

### 리눅스 (Debian 계열)

```bash
$ sudo apt update
$ sudo apt install golang-go
```
